Advanced STL-based dental model segmentation and visualization

Development Version - Application is being prepared

How to start the application:

  1. Open terminal in this directory
  2. Run: npm install
  3. Run: npm run dev
  4. Open your browser to the provided localhost URL

Features:

  • Load and visualize STL files
  • Automatic geometric segmentation (curvature, K-means, region growing)
  • Manual refinement tools (lasso, brush, select)
  • Interactive tooth manipulation (move, rotate)
  • Color-coded visualization
  • Export segmentation data

Technology Stack:

  • Vue.js 3 + TypeScript
  • Three.js for 3D rendering
  • three-mesh-bvh for fast interactions
  • Vite for development
